# 🏋️‍♂️ Fitness Flames

A professional, high-energy landing page for a gym/fitness brand. This project is fully responsive and designed to look great on all devices.

## ✨ Features
* Responsive Navbar: Smooth navigation with a mobile-friendly sidebar.
* Hero Section: Eye-catching design with a 'Join Us' call to action.
* Clean UI: Dark mode aesthetic with aquamarine accents.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Flexbox & Media Queries)
* JavaScript (For sidebar toggle)
